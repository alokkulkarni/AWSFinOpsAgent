# Break-even & payment options

- **No Upfront** — highest rate, zero capital, easiest to justify; default for a first commitment.
- **Partial Upfront** — middle ground between rate and capital outlay.
- **All Upfront** — deepest discount, but only worth it once you trust the baseline.

Rule of thumb: only recommend a **3-year / All-Upfront** commitment when trailing 60-day usage is
stable AND current coverage is below the recommended level. Otherwise prefer **1-year / No-Upfront**
and revisit after the next billing cycle.

Always show the payback period implied by the tool's upfront cost vs. its monthly saving — never
assert a break-even you did not compute from tool output.
